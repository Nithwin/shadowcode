# Contributing to ShadowCode

Thanks for contributing.

## Before Opening an Issue

- Confirm the issue reproduces on the latest source.
- Disable non-built-in extensions and retry.
- Search existing issues to avoid duplicates.

## Bug Reports

Please include:

- ShadowCode version or commit hash.
- Operating system and version.
- Reproduction steps.
- Expected and actual behavior.
- Relevant logs and screenshots.

## Feature Requests

Please describe:

- The problem you are trying to solve.
- Your proposed solution.
- Alternatives considered.

## Pull Requests

- Keep each PR focused and scoped.
- Follow coding guidelines used in this repository.
- Run build/typecheck before requesting review.
- Link the issue your PR addresses.

## Local Development

Common commands:

- `npm run watch-client-transpile`
- `npm run watch-client`
- `npm run watch-extensions`
- `./scripts/code.sh`

## Code of Conduct

Be respectful and constructive in all project interactions.
